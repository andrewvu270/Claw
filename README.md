# Claw Machine 

Claw machine game with an NFT-inspired theme

## How to Play

1. Press and hold the horizontal button (→) to move the claw right
2. Release to stop
3. Press and hold the vertical button (↑) to move the claw up
4. Release — the claw drops, grabs, and returns
5. Click a collected toy to add it to your collection

## Tech Stack

- React
- Vite
- Javascript
- Supabase
